# Markdown 区块
## Markdown 区块基本用法
Markdown 区块引用是在段落开头使用 > 符号 ，然后后面紧跟一个空格符号：
```markdown
> Hello, World!
> Hello, How are you?
> I'm fine!
```
显示效果:
> Hello, World!  
> Hello, How are you?  
> I'm fine!

## Markdown 区块嵌套
区块是可以嵌套的，一个 > 符号是最外层，两个 > 符号是第一层嵌套，以此类推：
```markdown
> 最外层
> > 第一层嵌套
> > > 第二层嵌套
```
显示效果:
> 最外层
>> 第一层嵌套
>>> 第二层嵌套
